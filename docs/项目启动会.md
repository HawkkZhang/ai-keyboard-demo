# AI键盘项目启动会

---

## 一、产品与品牌定位

### 1.1 产品主张

**"每个人的AI键盘"**

### 1.2 目标用户

18-30岁年轻用户群体

### 1.3 核心差异化

个性化定制能力——不仅是装扮维度的个性化，更是键盘功能本身的自定义能力。用户可以自主决定键盘"能做什么"，而不只是"长什么样"。

### 1.4 品牌Slogan

**"每个人的AI键盘，懂你所想，随心而变"**

### 1.5 五大宣传点

| 卖点 | 差异化 | 说明 |
|------|--------|------|
| **极致个性化** | 不只是换皮肤，而是自定义键盘功能本身 | 工具栏自由组合、预设模板（汽车销售、跨境电商、饭圈粉丝等）、用户定义"键盘能做什么" |
| **主动式智能** | 场景感知 + 智能推荐，AI主动帮你 | 评论区推荐"高赞回复"、电商搜索推荐"比价助手"、聊天场景推荐"表情包生成" |
| **自然语言控制** | 用说话的方式控制键盘 | "帮我换个可爱的皮肤"、"明天下午3点提醒我开会"、"帮我写个生日祝福" |
| **情感化体验** | IP形象 + 性格联动，键盘也有人设 | 元气少女/商务精英等形象，形象、皮肤、性格三位一体 |
| **个性化学习** | 从输入中学习，成为你的专属键盘 | 自动提取常用联系人/地址/卡号、记住高频表达、推荐算法优化 |

## 二、产品架构（从定位到落地）

### 横切层（贯穿所有功能）

| 层级 | 模块 |
|------|------|
| **个性化与定制化** | AI服务组件化、工具栏自定义、场景智能推荐、AI助手与装扮系统 |

### 三纵（相对独立的能力领域）

| 前端交互能力 | 模型能力 | 输入功能 |
|-------------|---------|---------|
| AI键交互 | 端云协同模型 | 智能联想与补全 |
| 智能体使用 | 主Agent路由 | 语音输入增强 |
| 问AI界面 | 智能体调度 | 个人信息快速调用 |
| 结果展示编辑 | 场景意图识别 | 智能提醒 |
| 场景推荐UI | 信息提取 | |

**架构说明：**
- **横切层**：贯穿所有功能的通用能力
- **纵向维度**：三个相对独立的能力领域，由不同工作组负责（分工见 `docs/specs/pm-a~pm-d`）
- **共计 21 个功能模块**：与 specs 四维度一致

## 三、具体工作

### 3.1 四个工作组

#### 3.1.1 前端交互组（pm-a，6个模块）

**负责人：** [待定]

**核心目标：** 高效、直观的AI能力使用体验

**模块清单：**
1. AI服务组件化框架
2. 工具栏自定义
3. 场景推荐UI
4. AI键交互
5. 智能体使用交互
6. 问AI交互界面

**关键依赖：**
- AI服务组件化框架是工具栏自定义的前置条件
- 场景推荐UI依赖模型组的场景推荐策略

---

#### 3.1.2 模型能力组（pm-b，5个模块）

**负责人：** [待定]

**核心目标：** 智能决策与个性化支撑，提供AI/ML后端能力

**模块清单：**
1. 主Agent意图判断与路由
2. 智能体调度系统
3. 个性化信息提取
4. 场景推荐策略
5. 装扮性格与模型联动

**关键依赖：**
- 主Agent路由是问AI功能的核心
- 智能体调度依赖主Agent的路由决策
- 场景推荐策略为前端组的场景推荐UI提供算法支撑

---

#### 3.1.3 输入能力组（pm-c，6个模块）

**负责人：** [待定]

**核心目标：** 提升输入效率与准确性，打造AI增强的输入体验

**模块清单：**
1. 智能联想与补全
2. 语音输入增强
3. 个人信息快速调用
4. 智能提醒
5. 端云协同大模型
6. 场景与意图识别

**关键依赖：**
- 端云协同大模型是基础设施，被多个能力依赖
- 场景识别能力驱动智能联想的触发时机

---

#### 3.1.4 装扮能力组（pm-d，4个模块）

**负责人：** [待定]

**核心目标：** 形成统一"键盘人设"，提升用户归属感与表达欲

**模块清单：**
1. AI助手与装扮系统
2. AI助手形象定制
3. AI助手性格定制UI
4. 键盘皮肤系统

**关键依赖：**

### 3.2 里程碑

**整体原则：** 四个维度并行推进，依赖关系见 3.3 节。优先级由项目负责人决定。

### 3.3 关键依赖关系

```
端云协同大模型
    ├─→ 智能联想与补全
    ├─→ 场景与意图识别
    └─→ 语音输入增强

AI服务组件化框架
    ├─→ 工具栏自定义
    └─→ 场景推荐UI

主Agent路由
    └─→ 智能体调度系统

场景与意图识别
    └─→ 场景推荐策略
        └─→ 场景推荐UI

个性化信息提取
    └─→ 个人信息快速调用
```
