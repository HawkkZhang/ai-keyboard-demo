# AI键盘功能拆解设计

> 基于《AI键盘目标人群及产品功能规划》PDF，按「三纵两横」架构重新拆解全部功能模块。

## 产品愿景

大模型时代，输入不再只是"字符解码"，而是意图表达。AI键盘的品牌主张：**每个人的AI键盘**。

- "每个人"——个性化：不仅是装扮维度，而是键盘功能的自定义能力
- "AI键盘"——独特的AI功能：输入效果全面通过AI增强 + 主动式AI交互

## 目标人群

高频表达的年轻用户——以社交沟通与内容发布为主要输入场景的Z世代年轻人（18-30岁）。

特征：AI接受程度高、有个性、爱表达、兴趣广泛。

## 功能架构总览

```
┌─────────────────────────────────────────────────────┐
│              个性化与定制化（横切层）                    │
│   AI服务组件化 · 工具栏自定义 · 场景智能推荐            │
│   AI助手与装扮系统 · 语音后处理定制                     │
├───────────────┬───────────────┬─────────────────────┤
│  前端交互能力   │   模型能力     │     输入功能         │
│               │               │                     │
│ · AI键交互     │ · 端云协同模型  │ · 智能联想与补全      │
│ · 智能体使用   │ · 主Agent路由  │ · 语音输入增强       │
│ · 问AI界面     │ · 智能体调度   │ · 个人信息快速调用    │
│ · 结果展示编辑  │ · 场景意图识别  │ · 智能提醒          │
│ · 场景推荐UI   │ · 信息提取     │                     │
├───────────────┴───────────────┴─────────────────────┤
│              基础设施（横切层）                         │
│   配置管理 · 数据分析                                  │

```

- **个性化横切层**在最上方，是产品核心差异化，渗透到三个纵向维度的每一个功能模块
- **三个纵向维度**各自独立、边界清晰，可由不同团队负责
- **基础设施横切层**在底部，为所有功能提供支撑

**合计 21 个功能模块**：个性化 5 + 前端 5 + 模型 5 + 输入 4 + 基础设施 2

## 一、个性化与定制化（横切层）

产品灵魂——"每个人的AI键盘"。不是独立模块，而是贯穿前端、模型、输入三个维度的能力。

### 1.1 AI服务组件化框架

| 项目 | 内容 |
|------|------|
| **功能描述** | 所有AI服务（助手、智能体、剪贴板、常用语等）统一为可插拔组件，提供标准接口规范，是自定义能力的底座 |
| **涉及维度** | 前端 + 模型 |
| **关键子功能** | 统一组件接口规范；现有功能组件化改造（助手、剪贴板、常用语等）；组件的注册/发现/加载机制 |

### 1.2 工具栏自定义

| 项目 | 内容 |
|------|------|
| **功能描述** | 用户自主决定工具栏放哪些AI服务入口，支持拖拽排序、添加/移除，提供预设模板快速上手 |
| **涉及维度** | 前端 |
| **关键子功能** | 拖拽排序；添加/移除组件；预设模板（普通用户、汽车销售、跨境电商、饭圈粉丝等）；工具栏布局自适应 |

### 1.3 场景智能推荐

| 项目 | 内容 |
|------|------|
| **功能描述** | 根据当前场景（评论区、电商搜索、聊天等）自动推荐合适的AI服务入口到工具栏，补充用户手动配置 |
| **涉及维度** | 前端 + 模型 |
| **关键子功能** | 场景到AI服务的映射规则；推荐结果与用户自定义的融合策略；推荐频次的衰减机制（减少打扰） |

### 1.4 AI助手与装扮系统

| 项目 | 内容 |
|------|------|
| **功能描述** | 提供多套预设IP形象供用户选择，IP形象联动键盘皮肤/主题风格；助手性格（活泼/严肃/幽默等）与IP形象绑定，性格映射到大模型Prompt，形成统一的"键盘人设" |
| **涉及维度** | 前端 + 模型 |
| **关键子功能** | 预设IP形象库；IP形象与皮肤/主题联动；性格模板定义；性格→Prompt映射；助手在交互界面中的视觉呈现 |

### 1.5 语音后处理定制

| 项目 | 内容 |
|------|------|
| **功能描述** | 语音输入结果支持多种风格后处理（娱乐向/效率向），用户可自定义风格 |
| **涉及维度** | 输入 + 模型 |
| **关键子功能** | 多种预设后处理风格；用户自定义后处理风格；风格切换快捷入口 |

## 二、前端交互能力

用户直接看到和操作的界面层。

### 2.1 AI键交互

| 项目 | 内容 |
|------|------|
| **功能描述** | 键盘区域新增AI按键，点击触发AI增强输入 |
| **关键子功能** | AI键位置/样式/动效；点击触发联想补全增强；按键状态管理（可用/加载中/不可用）；与其他按键的布局适配 |

### 2.2 智能体使用交互

| 项目 | 内容 |
|------|------|
| **功能描述** | 从工具栏启动某个智能体后，键盘区域的交互流程 |
| **关键子功能** | 智能体激活后的键盘状态变化；输入区域适配（是否展示专属输入引导/预设Prompt）；多轮交互的承接方式；退出智能体/切回普通输入的路径 |

### 2.3 问AI交互界面

| 项目 | 内容 |
|------|------|
| **功能描述** | 开放性指令入口，用户可自由表达意图 |
| **关键子功能** | 对话界面设计与实现；语音/文本双模输入；快捷入口（空格键语音等）；对话历史记录；启用/禁用配置 |

### 2.4 AI结果展示与编辑

| 项目 | 内容 |
|------|------|
| **功能描述** | AI产出内容的展示、编辑和上屏 |
| **关键子功能** | 生成结果展示界面；流式输出渲染；结果编辑后上屏到输入框；生成失败/超时的提示处理 |

### 2.5 场景推荐UI

| 项目 | 内容 |
|------|------|
| **功能描述** | 场景识别结果的前端呈现 |
| **关键子功能** | 智能体推荐卡片（单个/多个）；推荐位置自适应与动画效果；用户反馈收集（点击/忽略） |

## 三、模型能力

后端AI/ML能力，键盘"智能"的核心引擎。

### 3.1 端云协同大模型

| 项目 | 内容 |
|------|------|
| **功能描述** | 端侧 + 云端模型的基础能力层，为所有AI功能提供算力底座 |
| **关键子功能** | 端侧大模型选型与部署；云端大模型API对接；端云协同策略（延迟敏感走端侧、质量敏感走云端）；模型调用性能优化 |

### 3.2 主Agent意图判断与路由

| 项目 | 内容 |
|------|------|
| **功能描述** | 问AI的核心枢纽，分析用户自然语言意图，解析指令并路由到对应处理模块 |
| **关键子功能** | 意图分类与指令解析（内部指令如换肤/开关设置，外部指令如写作/检索/计算）；路由决策逻辑；多轮对话支持；指令执行与结果反馈 |

### 3.3 智能体调度系统

| 项目 | 内容 |
|------|------|
| **功能描述** | 管理和调度专用智能体，匹配用户任务到最合适的智能体 |
| **关键子功能** | 智能体注册与管理框架；专用智能体（写作达人、高赞朋友圈、祝福大师等，可扩展）；智能体匹配算法；通用大模型兜底处理 |

### 3.4 场景与意图识别

| 项目 | 内容 |
|------|------|
| **功能描述** | 自动识别用户当前的输入场景，驱动场景推荐和AI键行为 |
| **关键子功能** | 场景识别（评论区、电商搜索、聊天、内容创作等）；场景与AI服务的映射规则；意图置信度计算；基于用户使用频率的推荐优化 |

### 3.5 个性化信息提取

| 项目 | 内容 |
|------|------|
| **功能描述** | 从用户日常输入中提取关键信息，供后续快速调用 |
| **关键子功能** | 实体识别（人名/昵称/公司等）；关系抽取（亲属关系、生日等）；个人数据提取（卡号/地址/邮箱）；任务与待办识别；个人信息库存储 |

## 四、输入功能

与文字输入直接相关的能力，键盘本职工作的AI升级。

### 4.1 智能联想与补全

| 项目 | 内容 |
|------|------|
| **功能描述** | 大模型增强的输入联想，AI键点击后的核心体验 |
| **关键子功能** | 云端大模型联想结果；端侧大模型二遍解码结果；联想结果展示与上屏；触发时机优化；结果缓存机制 |

### 4.2 语音输入增强

| 项目 | 内容 |
|------|------|
| **功能描述** | 语音输入结果经过AI后处理，支持多种风格 |
| **关键子功能** | 多种后处理风格定义（娱乐/效率/语义修正等）；用户自定义后处理风格；风格切换快捷入口 |

### 4.3 个人信息快速调用

| 项目 | 内容 |
|------|------|
| **功能描述** | 输入过程中快速调用已提取的个人信息 |
| **关键子功能** | 拼音输入时的个人信息候选；语音输入时的实体识别与替换；指令输入时的信息自动填充；高频信息优先展示（调频机制） |

### 4.4 智能提醒

| 项目 | 内容 |
|------|------|
| **功能描述** | 基于从输入中提取的任务和事件，临近时间提醒用户 |
| **关键子功能** | 事件时间解析；提醒规则配置；提醒触发与展示方式；提醒管理（查看/编辑/删除） |

## 五、基础设施（横切层）

### 5.1 用户配置管理

| 项目 | 内容 |
|------|------|
| **功能描述** | 统一管理用户的各项配置，支持云端同步 |
| **关键子功能** | 配置数据结构设计；本地存储；云端同步；配置版本管理；导入/导出 |

### 5.2 数据分析与优化

| 项目 | 内容 |
|------|------|
| **功能描述** | 收集用户行为数据，驱动功能优化和个性化推荐 |
| **关键子功能** | 埋点方案设计；关键指标定义（功能使用率、AI点击率、智能体调用成功率等）；数据上报；A/B测试框架 |

## 模块依赖关系

```
个性化横切层:
  AI服务组件化框架 ──→ 工具栏自定义
  AI服务组件化框架 ──→ 场景智能推荐
  AI助手与装扮系统 ──→ 语音后处理定制（风格与人设联动）

前端交互:
  AI键交互 ──→ AI结果展示与编辑
  智能体使用交互 ──→ AI结果展示与编辑
  问AI交互界面 ──→ AI结果展示与编辑

模型能力:
  端云协同大模型 ──→ 主Agent意图判断与路由
  主Agent意图判断与路由 ──→ 智能体调度系统
  场景与意图识别 ──→ 场景智能推荐（横切层）
  个性化信息提取 ──→ 个人信息快速调用（输入功能）

输入功能:
  智能联想与补全 ←── 端云协同大模型（模型能力）
  个人信息快速调用 ←── 个性化信息提取（模型能力）
  智能提醒 ←── 个性化信息提取（模型能力）
```
