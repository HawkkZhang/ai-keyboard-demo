# UI/交互设计参与范围

> 从 specs 四维度中提炼，供 UI 设计师和交互设计师参考。对应文档：`pm-a-frontend-interaction.md`、`pm-b-model-capabilities.md`、`pm-c-input-capabilities.md`、`pm-d-appearance-system.md`

---

## 一、前端交互（pm-a）

### 1. AI 服务组件化框架

| 子功能 | 设计参与点 |
|--------|------------|
| 组件在界面上的展示形态 | 各组件（助手、智能体、剪贴板、常用语等）的图标、入口样式、统一视觉规范 |

### 2. 工具栏自定义

| 子功能 | 设计参与点 |
|--------|------------|
| 拖拽排序 | 拖拽手势、排序过程动效、放置反馈 |
| 添加/移除组件 | 添加入口、移除操作、确认流程 |
| 工具栏布局自适应 | 不同组件数量下的布局规则、分页/滚动方案 |
| 不同组件的大小入口样式 | 大入口、小入口的视觉与信息层级 |
| 预设配置模板 | 模板选择界面、模板预览、快速应用流程 |

### 3. 场景推荐 UI

| 子功能 | 设计参与点 |
|--------|------------|
| 智能体推荐卡片展示 | 单个/多个智能体的卡片样式、图标、名称、布局 |
| 推荐位置自适应与动画效果 | 推荐条展开/收起动效、与工具栏的融合位置 |
| 推荐结果与用户自定义配置的展示融合 | 推荐内容与自定义工具的视觉区分、信息层级 |
| 用户反馈收集 | 点击/忽略的交互反馈、负反馈生效的视觉提示 |

### 4. AI 键交互

| 子功能 | 设计参与点 |
|--------|------------|
| 点击交互逻辑 | AI 键位置、样式、动效；点击联想/补全的视觉反馈 |
| 点击推荐智能体交互 | 推荐智能体展开的交互流程、与候选区的衔接 |

### 5. 智能体使用交互

| 子功能 | 设计参与点 |
|--------|------------|
| 智能体触发后的键盘状态变化 | 键盘区域布局变化、输入区与工具栏的适配 |
| 输入区域适配 | 专属输入引导、预设 Prompt 的展示样式 |
| 多轮交互的承接方式 | 多轮对话的界面承接、历史展示、追问入口 |
| 不同智能体的差异化交互 | 各智能体（写作、祝福等）的专属界面与交互差异 |

### 6. 问 AI 交互界面

| 子功能 | 设计参与点 |
|--------|------------|
| 问 AI 入口 UI | 工具栏组件形式、光球/图标样式 |
| 问 AI 触发文本输入指令的输入框样式 | 输入框布局、占位符、发送按钮 |
| 问 AI 触发语音输入指令的样式 | 语音录制状态、波形/动效、松手发送 |
| 通过问 AI 调用个人信息的样式展示和交互 | 个人信息在结果中的展示、选择与上屏交互 |
| 问 AI 添加日程时的结果展示和交互 | 日程添加结果、确认/编辑流程 |

## 二、输入能力（pm-c）

### 1. 智能联想与补全

| 子功能 | 设计参与点 |
|--------|------------|
| 联想结果展示与上屏 | 候选词区域布局、普通候选与 AI 候选的视觉区分、点击上屏反馈 |

### 2. 语音输入增强

| 子功能 | 设计参与点 |
|--------|------------|
| 后处理切换快捷设置入口 | 风格切换入口位置、选择界面、当前风格标识 |

### 3. 个人信息快速调用

| 子功能 | 设计参与点 |
|--------|------------|
| 拼音输入时的个人信息候选 | 个人信息在候选区的展示样式、与普通候选的区分 |
| 个人信息编辑与管理界面 | 个人信息库的列表、编辑、删除、分类管理界面 |

### 4. 智能提醒

| 子功能 | 设计参与点 |
|--------|------------|
| 提醒展示方式 | 通知样式、键盘内提示的展示位置与样式 |
| 提醒管理 | 查看/编辑/删除/过期的管理界面、列表与详情 |

## 三、装扮能力（pm-d）

### 1. AI 助手与装扮系统

| 子功能 | 设计参与点 |
|--------|------------|
| 预设 IP 形象库 | 形象库的视觉设计、选择流程、预览 |
| IP 形象与皮肤/主题联动 | 形象与皮肤联动的展示效果 |
| 助手在交互界面中的视觉呈现 | 助手在问 AI、结果页等场景的展示样式 |

### 2. AI 助手形象定制

| 子功能 | 设计参与点 |
|--------|------------|
| AI 助手形象库设计 | 多种预设形象的视觉设计 |
| 助手形象在交互界面的展示 | 形象在各类界面中的展示规范 |
| 形象与皮肤/IP 联动 | 联动效果、切换时的视觉衔接 |
| UI 动画效果 | 形象相关的动效设计 |
| 形象切换动画 | 切换时的过渡动效 |

### 3. AI 助手性格定制（UI 部分）

| 子功能 | 设计参与点 |
|--------|------------|
| 预设多种性格模板的展示与选择 | 性格模板（活泼、严肃、幽默、专业等）的展示与选择界面 |
| 自定义性格描述输入界面 | 自定义输入框、示例、预览 |
| 性格与形象的绑定管理 UI | 绑定关系的展示、编辑、解绑流程 |

### 4. 键盘皮肤系统

| 子功能 | 设计参与点 |
|--------|------------|
| 预设皮肤库设计 | 皮肤库的视觉、分类、预览 |
| 皮肤与主题切换 | 切换入口、切换动效、当前状态标识 |
| 皮肤与 IP 形象联动 | 联动展示、组合选择界面 |

## 六、不涉及 UI/交互的模块

以下模块以算法、逻辑、后端为主，设计参与较少或无需参与：

- **模型能力**：端云协同大模型、智能体调度系统、个性化信息提取、场景推荐策略、装扮性格与模型联动（Prompt 映射等）
- **输入能力**：端云协同大模型、场景与意图识别（场景识别、意图判断等后端逻辑）
