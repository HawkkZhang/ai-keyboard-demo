# 装扮能力维度 — 功能描述

## 维度概述

负责用户个性化外观体验，以**换装**为核心：用户从预设形象与皮肤中选择，形成统一的「键盘人设」。本项目不做形象定制，性格与形象绑定，选择形象即获得对应性格，无需单独性格定制。性格与模型的联动能力由模型能力组负责。

---

## 1. AI 助手与装扮系统

**做什么：** 提供多套预设 IP 形象供用户选择（换装），IP 形象联动键盘皮肤/主题风格，形成统一的「键盘人设」。性格与形象绑定，选择形象即获得对应性格。

**关键子功能：**
- 预设 IP 形象库（形象选择，不支持定制）
- IP 形象与皮肤/主题联动
- 助手在交互界面中的视觉呈现与动效
- 问 AI 结果页的页面风格与选定形象适配
- 形象切换流程（选择形象即完成换装，性格随形象自动绑定）

---

## 2. 键盘皮肤系统

**做什么：** 提供多种键盘皮肤与主题，与 IP 形象和装扮风格联动。

**关键子功能：**
- 预设皮肤库设计
- 皮肤与主题切换
- 皮肤与 IP 形象联动
