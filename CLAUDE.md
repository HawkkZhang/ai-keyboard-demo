# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在本仓库中工作时提供指引。

## 项目概述

AI键盘 — 面向年轻用户（18-30岁）的AI增强输入法。产品愿景："每个人的AI键盘"，融合个性化定制、AI增强输入和主动式AI交互。

项目当前处于规划和 Demo 阶段，尚无构建系统、包管理器或测试框架。

## 运行 Demo

直接在浏览器中打开 `index.html`。这是一个自包含的单文件 Demo（约2500行），HTML、CSS、JS 全部内联，无需服务器或构建步骤。

## 架构

### "三纵两横"架构（与 docs/specs/pm-a~pm-d 一致）

- **横切层：** 个性化与定制化（AI服务组件化、工具栏自定义、场景智能推荐、AI助手与装扮系统）

- **纵向维度（各自独立）：**
  - 前端交互能力（6个模块）：AI服务组件化、工具栏自定义、场景推荐UI、AI键、智能体使用、问AI
  - 模型能力（5个模块）：主Agent路由、智能体调度、信息提取、场景推荐策略、装扮性格与模型联动
  - 输入功能（6个模块）：端云协同大模型、场景与意图识别、智能联想与补全、语音输入增强、个人信息快速调用、智能提醒
  - 装扮能力（4个模块）：AI助手与装扮系统、形象定制、性格定制、键盘皮肤系统

共计 21 个功能模块。

### Demo 结构（index.html）

单文件 Demo 按以下顺序组织：
1. **CSS 样式** — 手机框架、聊天UI、键盘布局、工具栏、AI键动效、生成卡片、提示面板
2. **HTML** — 手机框架（状态栏、聊天区域、带工具栏的键盘）、场景切换侧边栏
3. **JS 部分**（按顺序）：
   - 场景定义与主题数据
   - 工具栏自定义逻辑（拖拽排序、添加/移除组件）
   - 场景切换与聊天渲染
   - AI键交互（候选词栏、智能体推荐、直接生成）
   - 提示面板逻辑

关键 DOM ID：`chatArea`、`sceneBar`、`appTitleBar`、`aiSceneInput`、`aiGenCard`

关键 JS 全局变量/函数：`currentScene`、`AI_KEY_SCENES`、`switchScene()`、`renderToolbar()`、`showAIKeyResult()`、`resetAIKeyState()`
